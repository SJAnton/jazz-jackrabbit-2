
### Para crear el makefile, ir a la carpeta build y hacer:
```sh
cmake ..
```
### Luego, para compilar hacer:
```sh
make
```

### Se van a crear 2 archivos ejecutables.

### Para correr el servidor hacer:
```sh
./server 8080
```

### Para correr un cliente hacer:
```sh
./client localhost 8080
```
